NAME

equo-unusedpackages − look for unused packages (pay attention)

SYNOPSIS

equo unusedpackages [−h] [−−quiet] [−−sortbysize] [−−by−user]

INTRODUCTION

Report unused packages that could be removed.

OPTIONS

"equo unusedpackages" supports the following options which alters its behaviour.

OPTIONAL ARGUMENTS
−−help

show this help message and exit

−q

show less details (useful for scripting)

−−sortbysize

sort packages by size

−−by−user

include packages installed by user
